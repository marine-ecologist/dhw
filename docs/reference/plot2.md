# Simple XY plot for terra::rast values() comparisons

Helper function to quickly visualise the values of two terra::rast files
in a base r xy plot (with abline) Assumes one var per rast

## Usage

``` r
plot2(x, y)
```

## Arguments

- x:

  terra::rast(a)

- y:

  terra::rast(b)

## Value

base R plot
