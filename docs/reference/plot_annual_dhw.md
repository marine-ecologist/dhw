# Plot mean monthly maximum for a given gridcell

Plot mean monthly maximum for a given gridcell from a terra::rast file
input

## Usage

``` r
plot_annual_dhw(input, fixedyears = NULL)
```

## Arguments

- input:

  rast file

- fixedyears:

  range y lims for ggplot

## Value

ggplot of annual DHW scaled Nov-Aug
